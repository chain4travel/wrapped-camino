# Wrapped Camino (WCAM)

WrappedCAM is an ERC20-based token that represents wrapped CAM. This smart contract
allows users to deposit native CAM tokens and receive WCAM tokens in return.
Conversely, WCAM tokens can be redeemed (burned) in exchange for their equivalent
value in native CAM.
